# gpa-calculator
A Flask-based GPA (Grade Point Average) calculator that allows students to calculate semester GPA, store GPA history, and manage saved academic records with optional data deletion.


# Features:
Calculates GPA for a trimester (semester-based calculation).
Stores GPA history for different trimesters in a database.
Stores academic data (subjects, grades, credits).
Allows the user to delete saved records at will.
